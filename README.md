# Move Zeros To End

## Description
This project contains a Python function that moves all zero values in a list
to the end while maintaining the order of non-zero elements.

The project also includes automated tests written using **pytest**
to verify that the function works correctly for different scenarios.

---

## Function Overview

**Function name:** `MoveZerosToEnd`

### What it does
- Takes a list of integers
- Moves all `0` values to the end
- Keeps the order of non-zero numbers unchanged
- Returns the updated list
